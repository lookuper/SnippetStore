SnippetStore
============

SnippetStore is application for saving C# code snippets with syntax highlighting. 

Requirenments: 
  Win XP/7/8, 
  MS .NET Framework 4.0+
  
Features: 
  C# code syntax highlighting.
  Auto save window position.
  Tab control navigation.
